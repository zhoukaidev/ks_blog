## 什么是数据结构与算法

`数据结构与算法`是一个非常有意思的命题，当提到这个名词的时候，实际上我们在讨论两件不同的事情:

* 数据结构
    * 数据结构是指如何来存储内容
* 算法
    * 算法是指如何来操作这些存储的内容，比如`"查询"`，`"查找"`, `"删除"`,`"插入"`等操作。
